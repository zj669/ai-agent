## 2. 测试环境准备

### 2.1 数据库表结构
- `agent_info`: Agent 主表
- `agent_version`: Agent 版本表

### 2.2 测试数据准备
- 测试用户 ID: 1
- 测试 Agent 数量: 初始为空，测试过程中创建

### 2.3 API 端点
| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/agent/create` | POST | 创建 Agent |
| `/api/agent/update` | POST | 更新 Agent |
| `/api/agent/publish` | POST | 发布 Agent |
| `/api/agent/rollback` | POST | 回滚 Agent |
| `/api/agent/list` | GET | 查询 Agent 列表 |
| `/api/agent/{id}` | GET | 查询 Agent 详情 |
| `/api/agent/{id}/versions` | GET | 查询版本历史 |
| `/api/agent/{id}/versions/{version}` | DELETE | 删除指定版本 |
| `/api/agent/{id}/force` | DELETE | 强制删除 Agent |
